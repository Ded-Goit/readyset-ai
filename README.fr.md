# ReadySet.AI

Plateforme d'onboarding assistée par IA pour les équipes commerciales B2B.

ReadySet.AI aide les entreprises à réduire le temps d'intégration des nouveaux employés, améliorer la formation et fournir un accès instantané aux connaissances internes grâce à l'intelligence artificielle.

---

## 🚀 Fonctionnalités

- 🌍 Support multilingue (English / Ukrainian / Français)
- 🤖 Parcours d'intégration IA (30 / 60 / 90 jours)
- 🎯 Simulations de vente avec Training Arena
- 📚 Assistant IA basé sur les documents internes
- 📊 Signaux de risque pour les mentors
- ✨ Animations avec Framer Motion
- 📱 Design entièrement responsive
- ⚡ Développé avec Next.js App Router

---

## 🛠 Technologies

- Next.js 16
- React 19
- Framer Motion
- Context API
- CSS Modules
- JavaScript (ES6+)

---

## 🌍 Changement de langue

Langues disponibles :

- English (EN)
- Ukrainian (UA)
- Français (FR)

La langue choisie est enregistrée dans le LocalStorage.
