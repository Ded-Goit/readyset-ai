# ReadySet.AI

AI-платформа для онбордингу B2B-відділів продажів.

ReadySet.AI допомагає компаніям скорочувати час адаптації нових співробітників, покращувати навчання та надавати миттєвий доступ до внутрішніх знань компанії за допомогою штучного інтелекту.

---

## 🚀 Можливості

- 🌍 Підтримка трьох мов (English / Українська / Français)
- 🤖 AI-план адаптації 30 / 60 / 90 днів
- 🎯 Training Arena для симуляції продажів
- 📚 AI-помічник для пошуку внутрішніх знань
- 📊 Система сигналів ризику для менторів
- ✨ Анімації появи через Framer Motion
- 📱 Повністю адаптивний дизайн
- ⚡ Побудовано на Next.js App Router

---

## 🛠 Технології

- Next.js 16
- React 19
- Framer Motion
- Context API
- CSS Modules
- JavaScript (ES6+)

---

## 🌍 Перемикач мов

Підтримуються:

- English (EN)
- Українська (UA)
- Français (FR)

Вибрана мова автоматично зберігається в LocalStorage.
